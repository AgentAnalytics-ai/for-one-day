# Create Media Bucket in Supabase

The `media` bucket is missing. Create it now:

## Steps:

1. **Go to Supabase Dashboard**
   - Navigate to your project

2. **Open Storage**
   - Click **"Storage"** in the left sidebar

3. **Create New Bucket**
   - Click **"New bucket"** button
   - **Name**: `media`
   - **Public**: **OFF** (make it Private)
   - Click **"Create bucket"**

4. **Verify**
   - You should now see the `media` bucket in your Storage list

## That's it!

Once the bucket is created, re-run the verification script to confirm everything passes.
